# Food-Delivery-Website
Developed a responsive food delivery web application using React.js, implementing reusable components, React Hooks, and optimized state management for smooth user interactions and fast page rendering.
Integrated dynamic features such as restaurant listings, menu browsing, cart management, and order placement, using REST APIs to fetch real-time data and ensure seamless user experience.
Implemented clean UI/UX with modern styling (Tailwind/CSS), routing with React Router, and efficient data handling, improving overall performance, scalability, and maintainability of the application.
